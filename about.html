Ecommerce Course Project

Bookstore eCommerce Website - Computer Science Department

This project was developed as part of the Ecommerce Course at Unity University. It demonstrates practical application of web development, UI/UX design principles, and e-commerce functionalities.

Bootstrap 5 JavaScript HTML/CSS GitHub Pages

Project Technical Overview

Technical Implementation

  • Frontend: HTML5, CSS3, Bootstrap 5, JavaScript
  • Responsive Design: Mobile-first approach
  • State Management: LocalStorage for cart simulation
  • Third-party Integration: Google Forms, Google Maps
  • Hosting: GitHub Pages (Free static hosting)

Course Requirements Met

  • ✅ Responsive homepage with Bootstrap
  • ✅ Four categories with 20+ unique books
  • ✅ Detailed product pages with all information
  • ✅ JavaScript-based search and filtering
  • ✅ Functional contact form (Google Forms)
  • ✅ Add-to-cart simulation
  • ✅ Order form simulation
  • ✅ Complete project documentation
  • ✅ Live deployment

Project Development Timeline

Week 1: Project Selection & Proposal

Selected Project 4 (Bookstore), formed team, submitted project proposal document.

Week 2: Website Structure & UI/UX Design

Created sitemap, wireframes, and set up Bootstrap framework.

Week 3: Content Entry & Development

Added 20+ books, implemented core functionalities, and built all pages.

Week 4: Integration & Testing

Integrated Google Forms, tested all features, ensured responsiveness.

Week 5: Final Submission & Presentation

Prepared final documentation, created presentation, deployed website.

Key Features Implemented

Advanced Search

Real-time JavaScript filtering by title and author with instant results.

Shopping Cart

Full cart simulation with add/remove items, quantity control, and order summary.

Responsive Design

Mobile-first approach ensuring perfect display on all device sizes.
